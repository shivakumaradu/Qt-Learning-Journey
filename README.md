# Qt Learning Journey 🚀

Welcome to my Qt Learning Journey repository!

This repository contains a collection of mini projects that I developed while learning Qt and C++. The goal of these projects is to understand GUI development, event-driven programming, custom widgets, painting, mouse handling, and drag-and-drop functionality using the Qt Widgets framework.

---

# Technologies Used

* C++
* Qt Widgets
* Qt Creator
* QPainter
* Event Handling
* Object-Oriented Programming (OOP)

---

# Mini Projects

## Mini Project 1 – Simple Paint Application 🎨

### Overview

A basic paint application that allows users to draw freehand sketches on a canvas using the mouse.

### Features

* Freehand drawing
* Mouse press, move, and release handling
* Real-time drawing updates
* Clear canvas functionality
* Custom painting using QPainter

### Qt Concepts Learned

* QWidget subclassing
* paintEvent()
* QPainter
* Mouse Events

  * mousePressEvent()
  * mouseMoveEvent()
  * mouseReleaseEvent()
* Event-driven programming

### What I Learned

This project helped me understand how custom painting works in Qt and how mouse events can be used to create interactive graphical applications.

---

## Mini Project 2 – Rectangle Drawing Canvas 📐

### Overview

An interactive canvas that allows users to draw rectangles by clicking and dragging the mouse.

### Features

* Click-and-drag rectangle creation
* Rubber-band rectangle preview
* Drawing in any direction
* Rectangle normalization
* Minimum size validation

### Qt Concepts Learned

* QRect
* QRect::normalized()
* Real-time shape rendering
* Geometry handling
* Interactive graphics

### What I Learned

This project taught me how graphical editors provide live visual feedback while drawing shapes. It also improved my understanding of coordinate systems and rectangle geometry.

## Mini Project 3 – Drag & Drop Widget Builder 🖱️

### Overview

A simple GUI builder that allows users to drag widgets from a toolbox and drop them onto a design canvas.

### Features

* Drag-and-drop functionality
* Dynamic widget creation
* Design canvas
* Widget palette
* Interactive user interface

### Supported Widgets

* Button
* Label
* Text Field
* Check Box
* Radio Button

### Qt Concepts Learned

* Qt Drag & Drop Framework
* MIME Data
* Drag Events
* Drop Events
* Dynamic Widget Management
* Object-Oriented GUI Design

### What I Learned

This project helped me understand the internal concepts behind GUI designers and visual form-building tools. It also improved my knowledge of event handling and dynamic widget creation.

## Mini Project 4 – Canvas Widget Designer

### Overview

A mini visual GUI designer developed using C++ and Qt Widgets. Users can drag widgets from a toolbox, place them on a design canvas, select them, and reposition them interactively.

### Features

* Drag and Drop Widget Placement
* Interactive Design Canvas
* Widget Selection
* Widget Repositioning
* Grid Background
* Dynamic Widget Creation
* Multiple Widget Types
* Canvas Management
* Clear All Functionality

### Supported Widgets

* Button
* Label
* Text Field
* Checkbox
* Radio Button

### Qt Concepts Learned

* QWidget
* Custom Painting
* Drag & Drop Framework
* Mouse Events
* Event Filters
* Dynamic Widget Creation
* Object-Oriented Design
* Layout Management
* GUI Development

### What I Learned

This project helped me understand how GUI builders such as Qt Designer work internally. I learned how to dynamically create widgets, manage user interaction, handle drag-and-drop operations, and build an interactive design surface using Qt Widgets.
